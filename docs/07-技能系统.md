# 技能系统

技能 (Skills) 是按需加载的领域知识——以 Markdown 文件存储，仅在 LLM 需要时通过 `skill` 工具加载。

> 对应源码：`src/openharness/skills/`

---

## 模块组成

| 文件 | 职责 |
|------|------|
| `loader.py` | 技能加载：bundled → user → plugins |
| `registry.py` | `SkillRegistry` — 技能注册与查找 |
| `types.py` | `SkillDefinition` — 技能数据模型 |
| `bundled/` | 内置技能目录 |

---

## SkillDefinition 数据模型

```python
@dataclass
class SkillDefinition:
    name: str            # 技能名称（如 "commit", "review"）
    description: str     # 简短描述
    content: str         # 完整 Markdown 内容
    source: str          # 来源："builtin", "user", "plugin"
    path: str            # 文件路径
```

## 技能加载流程

> 源码：[`skills/loader.py`](../src/openharness/skills/loader.py)

```python
def load_skill_registry(cwd=None) -> SkillRegistry:
    registry = SkillRegistry()

    # 1. 加载内置技能 (bundled/)
    for skill in get_bundled_skills():
        registry.register(skill)

    # 2. 加载用户自定义技能 (~/.openharness/skills/*.md)
    for skill in load_user_skills():
        registry.register(skill)

    # 3. 加载插件提供的技能
    if cwd is not None:
        settings = load_settings()
        for plugin in load_plugins(settings, cwd):
            if plugin.enabled:
                for skill in plugin.skills:
                    registry.register(skill)

    return registry
```

三级加载优先级：`bundled` → `user` → `plugins`（后注册的同名技能覆盖先注册的）。

## YAML Frontmatter 解析

> 源码：`loader.py:_parse_skill_markdown()`

技能 Markdown 文件支持 YAML frontmatter 格式：

```markdown
---
name: commit
description: Create clean, well-structured git commits
---

# Commit Skill

## When to use
Use when the user asks to commit changes...

## Workflow
1. Review staged changes with `git diff --staged`
2. Write a clear commit message
...
```

解析逻辑：

```python
def _parse_skill_markdown(default_name: str, content: str):
    name = default_name
    description = ""
    lines = content.splitlines()

    # 尝试 YAML frontmatter (--- ... ---)
    if lines and lines[0].strip() == "---":
        for i, line in enumerate(lines[1:], 1):
            if line.strip() == "---":
                # 逐行解析 name: 和 description:
                for fm_line in lines[1:i]:
                    if fm_line.strip().startswith("name:"):
                        name = fm_line.strip()[5:].strip().strip("'\"")
                    elif fm_line.strip().startswith("description:"):
                        description = fm_line.strip()[12:].strip().strip("'\"")
                break

    # 回退：从标题和首段提取
    if not description:
        for line in lines:
            stripped = line.strip()
            if stripped.startswith("# "):
                name = stripped[2:].strip() or default_name
            elif stripped and not stripped.startswith("---"):
                description = stripped[:200]
                break

    return name, description
```

**兼容两种格式**：
1. YAML frontmatter（推荐）：结构化的 `name` 和 `description`
2. 纯 Markdown：从 `# 标题` 提取名称，从首个非空段落提取描述

## 用户技能目录

```python
def get_user_skills_dir() -> Path:
    return get_config_dir() / "skills"
    # 通常为 ~/.openharness/skills/
```

用户只需将 `.md` 文件放入此目录即可添加技能：

```bash
# 添加自定义技能
cp my-skill.md ~/.openharness/skills/

# 兼容 anthropics/skills 格式
cp anthropics-skill.md ~/.openharness/skills/
```

## 技能如何被使用

### 1. System Prompt 中列出可用技能

> 源码：`prompts/context.py:_build_skills_section()`

```python
def _build_skills_section(cwd):
    registry = load_skill_registry(cwd)
    skills = registry.list_skills()
    lines = [
        "# Available Skills",
        "",
        "The following skills are available via the `skill` tool. "
        "When a user's request matches a skill, invoke it with "
        '`skill(name="<skill_name>")` to load detailed instructions.',
    ]
    for skill in skills:
        lines.append(f"- **{skill.name}**: {skill.description}")
    return "\n".join(lines)
```

这部分会被注入到 system prompt 中，让 LLM 知道有哪些技能可用。

### 2. LLM 通过 `skill` 工具按需加载

> 源码：`tools/skill_tool.py`

LLM 在需要时调用 `skill(name="commit")` 工具，返回技能的完整 Markdown 内容作为上下文。

### 3. 工作流示例

```
用户: "帮我提交这些更改"
LLM: 检测到与 "commit" 技能相关
     → 调用 skill(name="commit")
     → 获取完整的 commit 工作流指引
     → 按照指引执行：git diff → git add → git commit
```

## SkillRegistry

```python
class SkillRegistry:
    def register(self, skill: SkillDefinition) -> None
    def get(self, name: str) -> SkillDefinition | None
    def list_skills(self) -> list[SkillDefinition]
    def search(self, query: str) -> list[SkillDefinition]
```

`search()` 支持关键词搜索，用于 `tool_search` 工具。

- 相同的 YAML frontmatter 格式
- 相同的 `name` / `description` 字段
- 只需将 `.md` 文件复制到 `~/.openharness/skills/` 即可使用
