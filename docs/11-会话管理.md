# 会话管理

会话管理系统负责对话历史的持久化、恢复和导出，支持跨会话的连续工作流。

> 对应源码：`src/openharness/services/session_storage.py` + `src/openharness/services/compact/`

---

## 核心功能

| 功能 | 函数 | 说明 |
|------|------|------|
| **保存** | `save_session_snapshot()` | 将当前会话保存为 JSON |
| **加载最新** | `load_session_snapshot()` | 加载最近的会话 |
| **列表** | `list_session_snapshots()` | 列出所有保存的会话 |
| **按 ID 加载** | `load_session_by_id()` | 加载特定会话 |
| **导出** | `export_session_markdown()` | 导出为 Markdown |

---

## 两种会话目录（重要区分）

OpenHarness 中存在两种不同的"会话"概念，路径完全不同：

### 1. 普通会话快照（session snapshots）

> 源码：[`services/session_storage.py`](../src/openharness/services/session_storage.py) + [`config/paths.py`](../src/openharness/config/paths.py)

每个项目有独立的会话目录，位于**用户数据目录**中：

```python
def get_project_session_dir(cwd):
    path = Path(cwd).resolve()
    digest = sha1(str(path).encode("utf-8")).hexdigest()[:12]
    session_dir = get_sessions_dir() / f"{path.name}-{digest}"
    return session_dir
```

**实际路径**：`~/.openharness/data/sessions/<project>-<hash>/`

路径示例：`~/.openharness/data/sessions/OpenHarness-a1b2c3d4e5f6/`

### 2. Agent Debug 会话

> 源码：[`agent_debug.py`](../src/openharness/agent_debug.py)

Agent debug 会话位于**当前工作目录**下：

```python
AGENT_SESSIONS_ROOT = Path(".openharness/sessions")
```

**实际路径**：`<cwd>/.openharness/sessions/<session-id>/`

> 这两种会话使用不同的存储机制和目录。不要混淆。

## 保存会话快照

```python
def save_session_snapshot(*, cwd, model, system_prompt, messages, usage, session_id=None):
    session_dir = get_project_session_dir(cwd)
    sid = session_id or uuid4().hex[:12]

    # 从首条用户消息提取摘要
    summary = ""
    for msg in messages:
        if msg.role == "user" and msg.text.strip():
            summary = msg.text.strip()[:80]
            break

    payload = {
        "session_id": sid,
        "cwd": str(Path(cwd).resolve()),
        "model": model,
        "system_prompt": system_prompt,
        "messages": [msg.model_dump(mode="json") for msg in messages],
        "usage": usage.model_dump(),
        "created_at": time.time(),
        "summary": summary,
        "message_count": len(messages),
    }

    # 同时保存两份
    latest_path = session_dir / "latest.json"    # 便于快速恢复
    latest_path.write_text(json.dumps(payload, indent=2))

    session_path = session_dir / f"session-{sid}.json"  # 按 ID 保存
    session_path.write_text(json.dumps(payload, indent=2))

    return latest_path
```

存储结构：
```
~/.openharness/data/sessions/OpenHarness-a1b2c3d4e5f6/
├── latest.json              # 总是指向最新会话
├── session-abc123def456.json  # 按 ID 保存
├── session-789012345678.json
└── transcript.md            # 导出的 Markdown（如有）
```

## 会话恢复

### 加载最新会话

```python
def load_session_snapshot(cwd):
    path = get_project_session_dir(cwd) / "latest.json"
    if not path.exists():
        return None
    return json.loads(path.read_text())
```

### 列出历史会话

```python
def list_session_snapshots(cwd, limit=20):
    session_dir = get_project_session_dir(cwd)
    sessions = []

    # 扫描所有 session-*.json 文件
    for path in sorted(session_dir.glob("session-*.json"),
                       key=lambda p: p.stat().st_mtime, reverse=True):
        data = json.loads(path.read_text())
        sessions.append({
            "session_id": data["session_id"],
            "summary": data.get("summary", ""),
            "message_count": data.get("message_count", ...),
            "model": data.get("model", ""),
            "created_at": data.get("created_at", ...),
        })
        if len(sessions) >= limit:
            break

    return sessions
```

### 按 ID 加载

```python
def load_session_by_id(cwd, session_id):
    session_dir = get_project_session_dir(cwd)
    path = session_dir / f"session-{session_id}.json"
    if path.exists():
        return json.loads(path.read_text())
    # 回退到 latest.json
    latest = session_dir / "latest.json"
    if latest.exists():
        data = json.loads(latest.read_text())
        if data.get("session_id") == session_id or session_id == "latest":
            return data
    return None
```

## Markdown 导出

```python
def export_session_markdown(*, cwd, messages):
    session_dir = get_project_session_dir(cwd)
    path = session_dir / "transcript.md"
    parts = ["# OpenHarness Session Transcript"]

    for message in messages:
        parts.append(f"\n## {message.role.capitalize()}\n")
        text = message.text.strip()
        if text:
            parts.append(text)
        # 工具调用
        for block in message.tool_uses:
            parts.append(f"\n```tool\n{block.name} {json.dumps(block.input)}\n```")
        # 工具结果
        for block in message.content:
            if getattr(block, "type", "") == "tool_result":
                parts.append(f"\n```tool-result\n{block.content}\n```")

    path.write_text("\n".join(parts))
    return path
```

## 上下文压缩

> 源码：`src/openharness/services/compact/`

当对话历史过长时，`/compact` 命令或自动压缩会将旧消息摘要化：

```
旧消息（详细对话） → LLM 摘要 → 一条摘要消息替代旧历史
```

这可以在保持上下文连贯的同时大幅减少 token 消耗。

## 设计要点

1. **双份保存**：`latest.json`（快速恢复）+ `session-{id}.json`（历史管理）
2. **SHA1 项目隔离**：不同项目路径的会话互不干扰
3. **JSON 完整快照**：包含完整消息链、模型信息、usage、时间戳
4. **自动摘要提取**：从首条用户消息提取前 80 字符作为会话摘要
5. **最多 20 条历史**：`list_session_snapshots(limit=20)` 避免列表过长
