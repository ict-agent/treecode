# 多智能体协调

OpenHarness 支持生成子代理（Subagent），管理后台任务，并通过团队注册表实现多智能体协调。

> 对应源码：`src/openharness/coordinator/` + `src/openharness/tasks/`

---

## 模块组成

### coordinator/ — 团队协调

| 文件 | 职责 |
|------|------|
| `coordinator_mode.py` | `TeamRegistry` — 团队创建、Agent 管理、消息传递 |
| `agent_definitions.py` | 内置 Agent 定义（如 reviewer, planner） |

### tasks/ — 后台任务

| 文件 | 职责 |
|------|------|
| `manager.py` | `BackgroundTaskManager` — 核心任务管理 |
| `types.py` | `TaskRecord`, `TaskStatus`, `TaskType` |

| `local_shell_task.py` | Shell 命令子进程生成 |
| `stop_task.py` | 任务停止逻辑 |

## 后台任务管理器

> 源码：[`tasks/manager.py`](../src/openharness/tasks/manager.py)

```python
class BackgroundTaskManager:
    """Manage shell and agent subprocess tasks."""

    def __init__(self):
        self._tasks: dict[str, TaskRecord] = {}
        self._processes: dict[str, asyncio.subprocess.Process] = {}
```

### 创建 Shell 任务

```python
async def create_shell_task(self, *, command, description, cwd, task_type="local_bash"):
    task_id = _task_id(task_type)
    output_file = get_tasks_dir() / f"{task_id}.out"

    record = TaskRecord(
        id=task_id,
        task_type=task_type,
        description=description,
        command=command,
        cwd=str(cwd),
        status="running",
        output_file=str(output_file),
        created_at=time.time(),
    )

    self._tasks[task_id] = record
    await self._start_process(task_id)
    return record
```

### 创建 Agent 任务

```python
async def create_agent_task(self, *, prompt, description, cwd,
                            task_type="local_agent", model=None, api_key=None, command=None):
    task_id = _task_id(task_type)

    # 构建 agent 执行命令
    if command is None:
        cmd_parts = ["uv", "run", "oh", "-p", shlex.quote(prompt)]
        if model:
            cmd_parts.extend(["-m", model])
        command = " ".join(cmd_parts)

    record = TaskRecord(
        id=task_id,
        task_type=task_type,
        description=description,
        command=command,
        prompt=prompt,
        cwd=str(cwd),
        status="running",
        ...
    )

    self._tasks[task_id] = record
    await self._start_process(task_id)
    return record
```

Agent 任务本质上是通过 `uv run oh -p <prompt>` 启动一个新的 OpenHarness 实例作为子进程。

### 进程启动

```python
async def _start_process(self, task_id):
    task = self._require_task(task_id)

    process = await asyncio.create_subprocess_exec(
        "/bin/bash", "-lc", task.command,
        cwd=task.cwd,
        stdout=asyncio.subprocess.PIPE,
        stderr=asyncio.subprocess.STDOUT,
        stdin=asyncio.subprocess.PIPE,
        env={**os.environ, "ANTHROPIC_API_KEY": task.api_key or ""},
    )

    self._processes[task_id] = process

    # 后台监控进程
    asyncio.create_task(self._watch_process(task_id, process, task.generation))
    asyncio.create_task(self._copy_output(task_id, process))
```

### 进程监控

```python
async def _watch_process(self, task_id, process, generation):
    """监控子进程退出，更新任务状态。"""
    returncode = await process.wait()
    task = self._tasks.get(task_id)
    if task and task.generation == generation:
        task = replace(task, status="done" if returncode == 0 else "error")
        self._tasks[task_id] = task
```

### 输出收集

```python
async def _copy_output(self, task_id, process):
    """将子进程 stdout 实时写入输出文件。"""
    task = self._require_task(task_id)
    with open(task.output_file, "a") as f:
        while True:
            line = await process.stdout.readline()
            if not line:
                break
            f.write(line.decode("utf-8", errors="replace"))
```

## TaskRecord 数据模型

```python
@dataclass
class TaskRecord:
    id: str               # 任务 ID（如 "agent_abc123"）
    task_type: TaskType    # "local_bash" / "local_agent" / "remote_agent" / "in_process_teammate"
    description: str       # 任务描述
    command: str           # 执行的命令
    cwd: str               # 工作目录
    status: TaskStatus     # "pending" / "running" / "done" / "error" / "stopped"
    output_file: str       # 输出文件路径
    created_at: float      # 创建时间戳
    prompt: str = ""       # Agent 任务的 prompt
    model: str = ""        # 使用的模型
    api_key: str = ""      # API Key
    progress: int = 0      # 进度 (0-100)
    status_note: str = ""  # 状态说明
    generation: int = 0    # 重启代数
```

### 停止任务

```python
async def stop_task(self, task_id):
    task = self._require_task(task_id)
    process = self._processes.get(task_id)
    if process and process.returncode is None:
        process.terminate()
        try:
            await asyncio.wait_for(process.wait(), timeout=5.0)
        except asyncio.TimeoutError:
            process.kill()
    self._tasks[task_id] = replace(task, status="stopped")
```

## 设计要点

1. **进程隔离**：每个 Agent 是独立的子进程，不共享内存
2. **输出持久化**：任务输出写入文件（`~/.openharness/tasks/{id}.out`）
3. **优雅停止**：先 SIGTERM，5 秒后 SIGKILL
4. **可重启**：支持 Agent 任务重启（generation 计数器跟踪）
5. **单例模式**：`BackgroundTaskManager` 和 `TeamRegistry` 都是全局单例
