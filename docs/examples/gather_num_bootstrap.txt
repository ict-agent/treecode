# Example `/execute` script: same topology as gather_handshake; each worker gets a unique integer 1–8.
#
# --- Canonical gather spec (git) vs runtime copy ---
#   Source of truth: docs/examples/gather_num.md
#   `/gather` loads:     .treecode/gather/gather_num.md  (project-local; often gitignored)
#   Keep them identical after edits:
#     mkdir -p .treecode/gather
#     cp docs/examples/gather_num.md .treecode/gather/gather_num.md
#
# --- Bootstrap (shared TUI / web console, main session) ---
#   /execute docs/examples/gather_num_bootstrap.txt
#
# Wait until spawned agents have finished starting.
#
# --- Then run gather (additive request = demo checks main 36, A 31) ---
# From main:
#   /gather --spec gather_num main@default "collect assigned numbers; subtree_value = sum of all assigned integers in this subtree (additive default)"
# From A only:
#   /gather --spec gather_num A@default "collect assigned numbers; subtree_value = sum of all assigned integers in this subtree (additive default)"
# Optional trace id:
#   /gather --gather-id my-run-1 --spec gather_num main@default "collect assigned numbers; subtree_value = sum of all assigned integers in this subtree (additive default)"
#
# This file only runs `/spawn` below; spec install + `/gather` are separate steps.
